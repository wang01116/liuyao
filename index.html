六爻起卦是一款传统六爻排盘工具，参考《增删卜易》等古籍构建。

🎲 起卦方式

  • 随机卦 · 数字起卦 · 手动指定（上爻→初爻下拉选取）

📊 卦象信息

  • 本宫卦伏藏爻列在爻的最左侧
  • 六神（红色）按日辰天干排布（龙/雀/勾/蛇/虎/玄）
  • 六亲按卦宫五行生克（父/兄/孙/财/官）
  • 世爻应爻；老阴×老阳○标注在本变之间
  • 变卦六亲以本卦卦宫五行为基准
  • 特殊卦：六冲/六合/游魂/归魂

🔍 地支分析

点击「地支分析」展开，流程图形式展示月日建对各爻的生克合冲刑关系、动爻变化十二长生等。

📅 干支历→阳历

起卦页面干支历下方「应期推算」功能，选择天干地支即可查找对应阳历日期区间。

📚 卦例管理

  • 命名保存，八类分类标签
  • 卦例下方可直接查看/编辑 Markdown 笔记
  • 导出/导入/单卦分享 JSON
  • GitHub Fine-grained Token 云端同步
☁ GitHub 同步配置
在 GitHub Developer settings → Fine-grained tokens 创建，需要目标仓库 Contents 读写权限。
