六爻起卦是一款传统六爻排盘工具，基于《增删卜易》等传统典籍的六爻体系构建。

🎲 三种起卦方式

  • 随机卦：随机生成六爻，含老阴(×)老阳(○)变爻，自动产生本卦与变卦
  • 数字起卦：输入三个1-9的数字，前两个分别决定上卦和下卦，第三个决定变爻位置
  • 手动指定：逐爻手动选择阴爻、阳爻、老阴(变阳)、老阳(变阴)

📊 卦象信息

  • 显示本卦与变卦的六爻排列、卦名、卦宫
  • 六神按日辰天干排布（甲乙→青龙起，以此类推）
  • 六亲按卦宫五行与地支五行生克关系确定
  • 世爻、应爻按卦宫序号规则标注
  • 显示伏藏爻（缺失六亲对应的本宫爻）
  • 变卦六亲仍以本卦卦宫五行为基准

📅 干支历

使用 lunar-javascript 开源库自动获取当前年月日时的干支，并计算空亡。

📚 卦例管理

  • 为每次起卦命名并保存，支持分类（工作/婚恋/求财等）
  • 每个卦例可添加 Markdown 格式笔记
  • 支持导出/导入 JSON，方便备份与分享
  • 支持通过 GitHub Fine-grained Token 同步数据

📱 PWA 支持

部署到 HTTPS 服务器后可安装为本地应用，支持离线使用。
