六爻起卦是一款基于传统六爻体系的排盘工具，参考《增删卜易》等古籍构建。

🎲 三种起卦方式

  • 随机卦：随机生成六爻含老阴(×)老阳(○)，自动产生本卦与变卦
  • 数字起卦：三个1-9数字分别决定上卦、下卦、变爻位置
  • 手动指定：下拉选择每爻的阴/阳/老阴/老阳

📊 卦象信息

  • 本卦与变卦横排并列显示，含卦名、卦宫五行
  • 六神按日辰天干排布（简写：龙/雀/勾/蛇/虎/玄）
  • 六亲按卦宫五行生克关系（简写：父/兄/孙/财/官）
  • 世爻、应爻标注；伏藏爻显示于对应爻下方
  • 特殊卦：六冲、六合、游魂、归魂
  • 变卦六亲以本卦卦宫五行为基准

🔍 卦象分析

  • 本卦变卦卦宫五行生克关系分析
  • 月日地支对各爻的生、克、合、冲、刑关系
  • 动爻与各爻的十二长生对应关系
  • 六合、三合局分析

📅 干支历

实时显示年月日时干支、节气、地支藏干、方位信息，空亡按日辰计算。

📚 卦例管理

  • 命名保存，支持八类分类标签
  • 每个卦例可添加 Markdown 笔记，笔记显示在卦象下方
  • 导出/导入/分享单卦 JSON
  • GitHub Fine-grained Token 云端同步
  • 批量删除、批量移动分类

📱 PWA
