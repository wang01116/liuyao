# 六爻起卦 · Liuyao Divination

一款基于传统六爻体系的在线排盘工具，参考《增删卜易》等古籍构建，支持 PWA 安装。

## ✨ 功能特性

### 三种起卦方式
| 方式 | 说明 |
|------|------|
| **随机卦** | 随机生成六爻，含老阴(×)老阳(○)变爻，自动产生本卦与变卦 |
| **数字起卦** | 三个 1-9 数字分别决定上卦、下卦和变爻位置 |
| **手动指定** | 下拉选择每爻：阴爻 / 阳爻 / 老阴（变阳）/ 老阳（变阴） |

**数字起卦映射规则：**
- 1/9 → 乾，2 → 兑，3 → 离，4 → 震，5 → 巽，6 → 坎，7 → 艮，8 → 坤
- 变爻：1/7→初爻，2/8→二爻，3/9→三爻，4→四爻，5→五爻，6→上爻

### 卦象显示
- 本卦与变卦**横排并列**显示（任意屏幕宽度均保持同行）
- 六神按日辰天干排布（简写：龙 / 雀 / 勾 / 蛇 / 虎 / 玄）
- 六亲按卦宫五行生克关系确定（简写：父 / 兄 / 孙 / 财 / 官）
- 变卦六亲仍以**本卦卦宫五行**为基准
- 世爻、应爻标注；伏藏爻灰色小字显示于对应爻下方
- 显示卦宫五行徽章及特殊卦名：六冲 / 六合 / 游魂 / 归魂

### 🔍 卦象分析
- **本变五行关系**：本卦与变卦卦宫五行生克分析（回头生主吉，回头克主凶等）
- **月建影响**：月地支对卦内各爻的生、克、六合、三合、冲、刑关系
- **日辰影响**：日地支对卦内各爻的生、克、六合、三合、冲、刑关系
- **动爻分析**：动爻与变爻之间的地支关系、十二长生
- **动爻互动**：动爻与卦内其他各爻的生克合冲刑关系
- **三合局**：卦内地支三合局检测

### 📅 干支历
- 实时显示年月日时四柱干支（天干地支按五行着色）
- 地支藏干参考（各地支的主气、中气、余气）
- 节气信息（当前节气和下一节气）
- 日辰空亡、日地支方位

### 📚 卦例管理
- 支持八类分类：💼工作 📖学业 💰求财 💕婚恋 🏠风水 🏥占病 🔮运势 📋其他
- 每个卦例含 Markdown 格式笔记，显示在卦象下方便于参考
- 批量删除、批量移动分类
- 导出/导入 JSON（支持分享单个卦例）
- GitHub Fine-grained Token 云端同步

### 🎨 其他
- 深色 / 浅色模式切换
- 干支五行配色（金黄 木绿 水蓝 火红 土棕）
- PWA 支持：部署到 HTTPS 后可安装为本地应用，支持离线使用
- 响应式设计，移动端友好

## 🚀 部署方式

### 静态托管（推荐）

将 `index.html`、`manifest.json`、`sw.js` 三个文件放入同一目录，部署到任意静态托管服务：

```bash
# GitHub Pages
# 1. 将文件推送到 GitHub 仓库的 gh-pages 分支
# 2. 在 Settings → Pages 中启用

# Vercel / Netlify
# 直接拖拽文件夹上传，或连接 GitHub 仓库
```

> **注意**：PWA 功能（Service Worker、安装提示）需要 HTTPS 环境才能正常工作。

### 本地预览

```bash
# Python 3
python3 -m http.server 8080

# Node.js npx
npx serve .
```

访问 `http://localhost:8080` 即可。

## 🔧 GitHub 同步配置

1. 在 GitHub → Settings → Developer settings → **Fine-grained tokens** 创建 Token
2. 选择目标仓库，授予 **Contents** 的读写权限
3. 在网站「☁ 同步」按钮中填入：
   - Token：`github_pat_...`
   - 仓库：`your-username/your-repo`
   - 文件路径：`records.json`（默认）
4. 点击「保存并推送」即可同步

### 六亲确定方式
以**卦宫五行**为基准，与各爻地支五行进行生克比较：
- 同宫五行 → 兄弟
- 宫生爻五行 → 子孙
- 爻生宫五行 → 父母
- 宫克爻五行 → 妻财
- 爻克宫五行 → 官鬼
